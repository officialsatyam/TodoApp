Agar User naya todo likhna chahta hai toh aapko user se 3 input aur lena hai jisme aap user se uska naam, aaj ka din , aur kya task vo likhna chahta ye sara data aapko as a json is POST api ke body me send karna hai data format ```{"name": "Sam", "day": "Wednesday", "task": "Today I will complete first task"}``` aisa hoga. Agar aapko server se status 201 mile toh iska matlb todo create ho gya hai. Ye baat aapko user ko btana hai terminal pe print karke. NOTE:
    POST api kaise call karte hai nodejs me request module use karke ye internet pe search
  4. Agar User todo delete karna chaahta hai to pahle aapko todo fetch karke terminal pe print krke user ko dikana hai uske baad user se input leke puchna hai ki vo konsa todo delete karna chaahta hai uske corresponding aapko todo ki id nikalni hai aur ye id aapko is DELETE https://learntodo.cyclic.app/delete/{todoId} api me daal kar call karna hai aur agar response me 200 aata hai toh aapko terminal pe todo delete ho gya print krke user ko inform karna hai.

Last But Not Least::::Existing todo ko yadi user edit karna chahta hai to aapko usko waapi todo fetch krke print krke user se input leke puchna hai vo konsa todo edit karn chaahta hai aur uske correspondig todo ki id nikal leni aur fir user se 3 input lena hai name, day aur task is data ko aapko kuch is tarah body me bhejna hai ```{"_id": "selectedid", "name": "Sam", "day": "Wednesday", "task": "Today I will complete first task"}``` is PUT https://learntodo.cyclic.app/edit api me. yadi response me 200 aaye to aapko terminal par todo update ho gya print krke user ko inform krna hai.
